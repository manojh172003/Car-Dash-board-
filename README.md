# Car Dashboard System

1. Description  
This project simulates a real-time car dashboard system using Embedded C.  
It displays important vehicle parameters such as speed, fuel level, gear position, and RPM.  
The system continuously updates values to mimic real-world dashboard behavior.

2. Features  
1. Speed monitoring  
2. Fuel level indication  
3. Gear position display  
4. RPM calculation  
5. Real-time data updates
